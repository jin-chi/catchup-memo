# `<<`（ヒアドキュメント）/ `<<<`（ヒアストリング）
- ヒアドキュメントは POSIX 標準機能
- ヒアストリングは Bash, Zsh の機能
- なのでヒアストリングは Dash を採用している Ubuntu では使えない
- ただし、`function`問題とは違い、メリットが大きいのでこれに関しては使用されることが多い

```bash
# ヒアストリング
sed '...' <<< "$var"  # echo をしなくても引数として渡せる
sed '...' <<< 'aaa'  # 文字列でも可

# ヒアドキュメント
# 入力サイズが大きい場合に特に役立つ
sed '...' <<EOF  # 変数はダブルクォートで囲まなくても展開される
$var
EOF

# 変数を展開したくない場合はキーをシングルクォートで囲む
sed '...' <<'EOF'
$var
EOF

# リダイレクトは間違えやすいので注意
sed '...' <<EOF > output.txt  # EOFの最後ではなく1行目にリダイレクトを記載する
aaa
bbb
ccc
EOF
```
