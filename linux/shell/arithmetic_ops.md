# Arithmetic Expansion（算術式展開）
Linux（主にbash）の算術式展開は、`$((式))`の構文を用いて整数計算を行い、その結果を式全体と置き換える機能。  
四則演算（`+`, `-`, `*`, `/`, `%`）、冪乗（`**`）、ビット演算、カッコによる優先順位変更が可能で、変数内の計算やインクリメントにも対応している。  
- 用途：主にシェルスクリプト内の数値計算に使用される。
- 制約：整数計算のみ。小数（浮動小数点）を扱う場合は`bc`コマンドを使用する必要がある。

## 1. `$(( ))`の基本：値の置き換え
`$((...))`は、その中に書かれた計算を行い、結果の数値に置き換わる。
```bash
echo "1 + 1 は $(( 1 + 1 )) です。"  # -> 1 + 1 は 2 です。
```

## 2. 主なお特徴

### ① 変数への代入
`(( ))`内でも代入は可能だが、計算結果を別の変数に入れたい場合は`$(( ))`が使われることが一般的。  
代入であることを強調したい時は`$`ありを使う方がいい。
```bash
total=$(( price * tax ))
```

### ② 変数に`$`が不要
`(( ))`と同様に、カッコ内の変数には`$`をつける必要がない。  
（つけても動くがつけない方が読みやすい）
```bash
num=5
result=$(( num * 2 ))  # $num と記載しなくていい
```

### ③ 実行結果の「コマンド」としての利用は不可
`$(( ))`は値に置き換わるだけなので、単体で1行で記載するとエラーになる。
```bash
$(( 1 + 1 ))  # ただの 2 に置き換わるのでエラーになる
```

## 3. 三項演算子との組み合わせ
算術式展開の中で三項演算子を使用できる。
> 構文：$(( 条件 ? 真の値 : 偽の値 ))
```bash
score=60
is_pass=$(( score >= 60 ? 1 : 0 ))
echo $is_pass  # -> 1
```

```bash
# 構文
result=$((1 + 2))
echo $((10 * 5))

# 算術式展開内では、変数に $ をつけなくても評価される
result=$((val1 + val2))
```

# 注意点：算術式展開は文字列に使用しない
- 文字列で式を指定した場合、算術式展開の再帰的処理の影響で、予期せぬ結果になるので基本的には数値に対して使用するようにする。
- 文字列で式を変数に代入し、その変数を算術式展開に指定すると、算術式展開の仕様でその変数の文字列が式である場合に式として認識し、計算してから変数の位置に置き換える動作となる。
- 理解して使う分には問題ないかもしれないが、そのようなパターンはほぼないと考えられし、そのような状況にならないように事前に値だけに整形するようにして、計算するようにすること。
```bash
# 文字列として代入した式が先に計算され、その後、算術式展開の計算が行われる
val1="value=200 + 100"
echo $(( value * 2 )) # -> 600
```

# 算術式評価
- 算術式評価は値を返さない。
- 終了ステータスを返す
- 数値の条件判定の場合に最初に検討する評価式
- `[ ... ]`よりも`((...))`の方が可読性がいいので推奨される

## 1. 変数の値を更新する（インクリメントなど）
ループのカウンターを増やす際によく使われる。
```bash
i=0
(( i++ ))      # i を1増やす
(( i += 10 ))  # i に10足す
```

## 2. 数値の比較
`if`の中で、`[ ]`や`[[ ]]`を使うよりも直感的に数値を比較できる。  
算術式評価が使えるBashなどの環境であれば、数値の比較は算術式評価を使うべき。
```bash
if (( a > 10 )); then
  echo '10より大きいです'
fi
```

## 3. `for`ループの制御
C言語スタイルのループを記述する際に使用する。
```bash
for (( i=0; i<5; i++ )); do
  echo $i
done
```

# 算術式展開と算術式評価の比較

## 違い
一番大きな違いは値を返すかどうか。
| 構文 | 名称 | 主な用途 | 特徴 |
| --- | --- | --- | --- |
| `(( ))` | 算術式評価 | 処理・更新・比較 | 値を返さない。C言語に近い書き方で操作ができる。 |
| `$(( ))` | 算術式展開 | 値の取得・代入 | 計算結果を文字列としてその場に置き換える（展開する）。 |

## 使い分け
| やりたいこと | 使う構文 | 例 |
| --- | --- | --- |
| 数値を表示したい | `$(( ))` | `echo $(( a + b ))` |
| 計算結果を代入したい | `$(( ))` | `result=$(( a * b ))` |
| ループのカウンタを増やしたい | `(( ))` | `(( i++ ))` |
| 数値を比較して分岐したい | `(( ))` | `if (( a  > b )); then` |
| 計算して「0」をコマンドとして実行したい | （不可） | ※どちらもエラーや意図しない挙動になる |
