# シングルクォーテーション
- シングルクォーテーションで囲んだ文字列はそのままの文字として認識される。
- なのでエスケープなどは必要なくなる。
- また、環境変数も展開されないので注意。
```bash
echo '\n' '\d' '\|'
```
- ダブルクォーテーションで囲んだ文字列は特殊文字などが認識されるので、特殊文字をそのままの文字列として使用したい場合は`\`エスケープが必要
- 逆に`\n`や`\d`などの特殊文字を特殊文字として扱いたい場合は`\\n`や`\\d`とする必要がある
- シングルクォーテーションの場合は`\n`, `\d`で良い
```bash
echo "\\n" "\\d" "|"
```
