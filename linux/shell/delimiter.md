# `--` (End of Options Delimiter)
- エンド・オブ・オプション・デリミタ (End of Options Delimiter)
- 他にも、ダブルダッシュ (Double Dash)、オプション区切り文字 (Option Delimiter)と呼ばれる
- オプションの終端を示すデリミタ。つまり、これ以降は引数であると示す。
- 例えば、`-a`という引数を指定した場合、コマンドによってはオプションと区別できないことがあるので、`-- -a`とすることで、`-a`は引数だと示すことができる

---

# 基本
```bash
# オプションと間違えないように、これは引数であると示す
grep -- '-a' app.log
```

---

# 応用

## 実行スクリプトのファイルパスを取得する
必須ではないが、万が一`${BASH_SOURCE:-$0}`ハイフン（`-`）などの特殊文字で始まる場合、意図していない動作や実行結果になる可能性があるためつけておくことが推奨される。

```bash
file_path=$(cd $(dirname -- ${BASH_SOURCE:-$0} && pwd))
```
