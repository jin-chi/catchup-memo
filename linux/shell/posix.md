# POSIX
UNIX 系 OS 全体の共通基準。  
POSIX に準拠した OS や shell であれば、準拠しているコマンドや仕様を共通で使用することができる。  
ただし、ブラケット（`[]`や`[[]]`）の中で使用する必要があるので注意。（一般的な正規表現と同様）

# コマンド

## POSIX 文字クラス
Bash や多くの Linux コマンド（`grep`, `awk`, `sed`など）で共通して使えるキーワード。  
正規表現を使わずに英数字やスペースなどを指定することができる。

| 文字クラス | 対象となる文字 | 正規表現でのイメージ |
| --- | --- | --- |
| [:alnum:] | 英数字 (Alphabet + Number) | [a-zA-Z0-9] |
| [:alpha:] | 英字 (Alphabet) | [a-zA-Z] |
| [:digit:] | 数字 (Digit) | [0-9] |
| [:lower:] | 英小文字 | [a-z] |
| [:upper:] | 英大文字 | [A-Z] |
| [:blank:] | スペースとタブ | [ \t] |
| [:space:] | 空白文字全般 (改行、垂直タブ等含む) | [\s] |
| [:cntrl:] | 制御文字 | （バックスペースなど） |
| [:graph:] | 印字可能文字 (空白は含まない) | [\x21-\x7E] |
| [:print:] | 印字可能文字 (空白を含む) | [\x20-\x7E] |
| [:punct:] | 記号 (Punctuation) | [!"#$%&'()*+,-./...] |
| [:xdigit:] | 16進数の数字 | [0-9a-fA-F] |

```bash
ls *[[:digit:]]*

# extglob と併用可能
ls !(*[[:digit:]])
```
