# printf
- Linux の組み込みコマンド
- `echo`より処理速度も速いのと、`echo`は環境によって末尾に改行が入ったり入らなかったりと予期せぬ挙動になる可能性があるので、ローカルで手軽に使う分には問題ないが、スクリプトなどでツールにしたり本番環境に入れる場合には`printf`を使用した方が良い。
- `printf`は組み込みコマンドと外部コマンドの2種類が存在していることを頭に入れておく。
- ただし、デフォルトでは組み込みコマンドが使用されるため、明示的に指定しない限りは外部コマンドは使用されない。


---

# 基本
- `echo`と違い末尾に改行が入らないので可読性を上げるために`\n`を入れることが多い
- パラメータが複数ある場合はパラメータ分の引数をまとめて指定することができる
- パラメータより引数が多い場合は、引数を処理し切るまでループする
```bash
printf '%s\n' "$var"  # デフォルトで改行が入らないので入れるようにする

# 以下の記述方法は変数に % が入った場合にエラーとなるのでしないこと
printf "$var"

# 複数の引数を指定
printf '%s %s %s %s\n' 'aaa' 'bbb' 'ccc' 'ddd'
# -> aaa bbb ccc ddd

# パラメータより引数の方が多い場合はループする
printf '%s\n' 'aaa' 'bbb' 'ccc' 'ddd'
# -> aaa
# -> bbb
# -> ccc
# -> ddd
```

## -v オプション
出力する代わりに変数に格納するオプション。  
`ksh`, `bash`, `zsh`でしか使えないオプションなので、他のシェルの場合は`bash`などに切り替えて使用する必要がある。

```bash
# 変数 var に代入する
printf -v var 'aaa'  # var=aaa

# フォーマットを指定して代入も可能 (こちらの方がコマンド置換で変数に入れるより軽い)
printf -v var '/home/%s/%s.txt' 'user''example'  # var=/home/user/example.txt

# スクリプトの場合は先頭で宣言する
#-> #!/bin/bash

# bash を指定して実行も可能
bash -c "printf -v var 'aaa'"
```

# 応用

## 数値を0埋めする
```bash
printf '%05d\n' 3  # -> 00003
```

## 3桁ごとのカンマ区切り
```bash
# 整数を3桁ごとのカンマ区切りにする
printf "%'d\n" 123456789  # -> 123,456,789

# 小数点を含む場合
printf "%'f\n" 123456.789  # -> 123,456.789000
```

## 日時の取得
`date`コマンドを使用するより処理が軽くて速い。古い bash(4.2未満)では使えない。  
また、`date`コマンドはナノ秒まで扱えるが、こちらは秒まで。  
ちなみに、どちらもカーネルのシステムクロックの時間を参照しているので、精度は変わらない。
```bash
# %(...)T の構文
printf '%(%Y-%m-%d %H:%M:%S)T\n'  # -> 2026-02-15 20:41:27

# 第二引数を指定する (bash5.0以降は第二引数を省略できる。デフォルトは-1)
printf '%(%Y-%m-%d %H:%M:%S)T\n' -1  # 現在日時
printf '%(%Y-%m-%d %H:%M:%S)T\n' -2  # スクリプト起動日時

# 以下のパラメータで同じ出力ができる
printf '%(%F %T)T\n'  # -> 2026-02-15 20:41:27

# timestamp のフォーマット変更が可能 (ただし timestamp 以外からは date コマンドの方が良い)
printf '%(%F %T)T\n' 1771156037  # -> 2026-02-15 20:47:17
```

## スペース区切りで文字列を指定すると順番に処理する
```bash
printf '%d.' 0xC0 0xA8 0x01 0x01
# -> 19216811
```

## 0埋め
```bash
printf '%02X\n' 1
# -> 01
```

## 10進数、16進数 の変換（IPアドレス）

### 10進数を16進数に変換する
- `.`をスペースに置換して展開し`printf`に渡す
- `x`または`X`で10進数を16進数に変換できる。`-x`が小文字、`-X`が大文字で出力できる。
- `02`は桁数と0埋め
```bash
IP_ADDRESS=192.168.1.1
printf '%02X %02X %02X %02X\n' ${IP_ADDRESS//./ }  # <- バラバラの引数として渡したいのでクォートで囲まない
# -> C0 A8 01 01
```

### 上記の16進数を10進数に戻してIPアドレスを確認する
- `printf`に値が16進数だとわかるように、先頭に`0x`を付与する必要がある
- `0x`をつけた値を`%d`で出力することで10進数になる
1. スペースを削除して変数を展開し`sed`に渡す
2. `sed`で2文字ずつ抜き出して先頭に`0x`を付与して末尾にスペースを入れてスペース区切りにする
3. `sed`の結果を`%d`で出力することで10進数になる
```bash
HEX_IP='C0 A8 01 01'
printf '%d.%d.%d.%d\n' $(sed -E 's/(..)/0x\1 /g' <<< "${HEX_IP// /}")
# -> 192.168.1.1
```
