# cut
文字列から特定の値を切り出すコマンド。  
区切り文字で分割して、切り出したいフィールド番号を選択する方法をよく使う。

# 基本仕様

## -d オプション
区切り文字を指定するオプション。
```bash
cut -d',' input.txt

# タブや改行などの特殊文字は $ をつける
cut -d$'\t' input.txt
```

## -f オプション
切り出すフィールド番号を指定するオプション。
```bash
# 2番目のフィールドを出力
cut -d',' -f2 input.txt

# 1番目と3番目のフィールドを出力
cut -d',' -f1,3 input.txt
```

## -c オプション
切り出す文字数を指定するオプション。

```bash
# 1〜3番目のフィールドを出力
echo 'hoge.prod.co.jp' | cut -c 6-3  # -> prod
```
