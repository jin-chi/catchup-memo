# date
日付や日時を扱うコマンド。

# 基本仕様
```bash
# locale の設定次第で英語表記や日本語表記が変わる
date  # ja_JP.UTF-8 -> 2026年 2月10日 火曜日 01時06分51秒 JST
      # en_US.UTF-8-> Tue Feb 10 01:09:05 JST 2026

# フォーマットを指定
date '+%Y%m%d'  # -> 20260210

# 特定の日時のフォーマットを整形
date '2026-02-10' '+%Y/%m/%d'  # -> 2026/02/10

# 年月が以下のフォーマットで出力可能
date "+%F"  # -> 2024-01-01

# unixスタンプ形式で表示
date +%s  # -> 1770739109
```

## -d (--date) オプション
```bash
# 昨日
date -d '1 day ago' '+%Y/%m/%d'

# 先月
date -d '1 month ago' '+%Y/%m/%d'
```

# 応用

## UNIXタイムスタンプを指定する場合
```bash
# @ 以降はunixタイムスタンプである、と明示的に示すための記号
# unixスタンプで指定してもよし、%Y-%m-%d の形式で指定してもよし
date --date "@${hoge_datetime}" "+%Y-%m-%d"
```
