# read
while文などでも使用されている。
標準入力から一行読み取るコマンド。

---

# 基本
```bash
# 基本構文
while read file; do
  echo $file
done < input.txt
```

## 複数の変数を指定
- 変数を複数指定することで、読み込むファイルのフィールド毎に値を取得することができる
- インプットファイルに対して変数の数が多い場合は`NULL`値が入る。逆に変数が足りない場合は足りない文のフィールドが取得されずに無視される。
- 区切り文字のデフォルトは空白またはタブ。区切り文字は明示的に指定することで変更することができる。
```bash
while read value1 value2 value3; do
  echo "$value1 $value2 $value3";
done < input.tsv
```

## `-r`オプション
- `-r`をつけることで`\n`などのエスケープシーケンスが値に含まれている場合にエスケープシーケンスの処理を無効にして文字列として扱うことができる
- `-r`をつけない場合、`\n`は改行と解釈されて予期せぬ挙動になってしまう可能性がある。
- つけるつけないは取得する値によって慎重に判断すること。つけておいた方が無難かも
```bash
while IFS=',' read -r value1 value2 value3 ...
  < input.txt
```

## `-sp`オプション
- 標準入力をつけつけて変数に格納
- `password`という変数に入力した文字が格納される
- `-p`オプションで標準入力の際に指定した文字列を表示することができる。（上記のコマンド例の`"Enter password: "`の部分）
- `-s`オプションで入力した文字を非表示にすることができる。パスワードなど目隠ししたい場合などに使用することが多い。
```bash
read -sp "Enter password: " password
```
