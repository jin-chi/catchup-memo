# rsync

---

# 基本
指定したファイルまたはディレクトリを最終引数に指定したディレクトリまたはリモートサーバーに同期する。
- `-a`オプションは属性情報もそのままコピーする
- `-v`は処理状況を出力する
- `-R`は再帰的にコピーする
```bash
rsync -avR src/dir1/file1 src/dir2/file2 dest/
```

## リモートサーバーにもコピー可能
-> `rsync`は`scp`よりも優れたコマンドではある。シンプルなファイルやディレクトリを転送するだけなら`scp`でも特に問題ない。
```bash
rsync -avR src/dir1/file1 user@remote:/path/to/dest/
```
