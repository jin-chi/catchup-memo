# 基本

## `-d`オプション
- カレントディレクトリの階層より下に深ぼらない。
```bash
ls -l my-dir  # my-dir 内のファイルやディレクトリが出力される
ls -ld my-dir  # my-dir 自体が出力される
```

## `-t`オプション
- 更新日の降順で表示する。
```bash
ls -t
```

## `-1`オプション
- `ls -l`のファイル名またはディレクトリ名だけが出力されるオプション。属性等の情報は表示されない。
- 先頭に total 表示も出ないので、実際のファイルやディレクトリの数を数える際には便利かも
```bash
ls -1
```
